Exercise 2: Windows Command Line Operations

Task Overview

  1. File Creation: Create myTextFile.txt with specified content
  2. Text Extraction: Extract lines containing "president" or "federal"
  3. File Comparison: Compare original and extracted files

Key Windows Commands Used

File Operations:

  • echo - Create file content
  • type - Display file contents
  • del - Delete files

Text Processing:

  • findstr - Search text patterns
  • fc - Compare files
  • /i - Case-insensitive flag

@echo off echo Creating myTextFile.txt...
